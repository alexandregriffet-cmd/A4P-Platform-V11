# A4P Sprint 2

Objectif : flux métier minimum club.
- Création équipe
- Ajout joueurs
- Génération passation CMP
- Formulaire joueur
- Enregistrement des résultats
- Tableau de collecte

## Ordre d'exécution
1. Exécuter `database/sprint2.sql`
2. Copier `.env.example` vers `.env.local`
3. Vérifier `lib/supabaseClient.ts` et `lib/supabaseServer.ts`
4. Créer équipe
5. Ajouter joueurs
6. Créer passation CMP
7. Ouvrir le lien joueur `?play=1`
8. Vérifier les réponses dans `/teams/[teamId]`
